# Zid-ZimonGame

Detta repository är startpunkten för vårt gemensamma Unity-spelprojekt!

## Syfte
Vi, som bröder, har skapat detta repo för att tillsammans utveckla ett helt nytt spel i Unity. Här samlar vi all kod, resurser och dokumentation som behövs för att ta projektet från idé till färdig produkt.

## Kom igång
1. Klona detta repo:
   ```sh
   git clone https://github.com/Nomiz1/Zid-ZimonGame.git
   ```
2. Öppna projektet i Unity Hub och välj rätt Unity-version.
3. Börja skapa, koda och ha kul!

## Struktur
- `Assets/` – Alla spelresurser, scripts, scener och prefabs.
- `ProjectSettings/` – Unitys projektinställningar.
- `.gitignore` – Ignorerar filer som inte ska versionshanteras (t.ex. Library, .DS_Store).

## Regler
- Commita ofta och skriv tydliga commit-meddelanden.
- Använd branches för nya features eller större ändringar.
- Respektera .gitignore så vi slipper onödiga filer i repot.
